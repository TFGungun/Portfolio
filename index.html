Demo Completed

Nix: Ashes of The Phoenix (Steam Early Access)

A project built with the great team at Doomsday Device Studios. Nix is a rogue-like that fancies the gamers that like to wreck the enemies completely, burning them, exploding them, utterly annihilating them. Nix also features the story of the Keepers, saving the world from creatures that intend to cause its demise.

Demo Completed

Math Titans (Playstore)

Passion project that started in 2019 finally coming to fruition. It is still the same multiplication game with Turn-Based RPG, but with design focused on how players can learn multiplication concept without memorization. I teamed up with an experienced educator in order to design the best mechanics for the students. As a result, more than 90% of the students playing the game (in the User Testing Phase) experienced an increase in their multiplication skill. But of course there are still a lot to improve, and with the acquired data, this game has been scheduled to be better.
